heumRolling

heumRolling은 롤링할 아이템들의 넓이와 상관없이 자동으로 아이템의 넓이를 구하여 자동으로 롤링해줍니다

파일다운로드

Basic

가장 기본 입니다.
그룹 사이즈=(아이템의 전체 넓이 - 마지막 아이템 넓이)

<script>
        $(window).load(function(){
                $("#button-data-rolling").heumRolling();
        });
</script>
<div id="button-data-rolling">

        </ul>
</div>

                });

                });

                });
        });
</script>
<div id="button-javascript-rolling">
        <ul class="heum-item-group">
                <li class="item"><img src="/resources/images/rolling/facebook.gif"/></li>
                        ....more image....
        </ul>
        <button type="button" id="pause">pause</button>
        <button type="button" id="start">start</button>

Events

Event Description
$(element).heumRolling().start() 롤링을 시작합니다
$(element).heumRolling().pause() 롤링을 멈춥니다
$(element).heumRolling().next() 다음 아이템으로 롤링합니다
$(element).heumRolling().prev() 이전 아이템으로 롤링합니다

Option

해당 옵션은 초기에 롤링 그룹을 설정해줍니다

Name Type Default Description
scroll String horizontal 세로 또는 가로를 지정해줍니다.
horizontal(가로)
vertical(세로)
rolling String next 롤링 방향을 지정해줍니다.
next(다음)
prev(이전)
autoscroll boolean true 초기에 스크롤을 할지의 여부입니다
button boolean false 버튼을 생성할지 말지의 여부입니다
timer Integer 5 움직임의 시간을 초단위로 조절합니다.
groupsize Integer|String null 롤링 그룹의 크기를 지정합니다. 만약 값이 그룹 사이즈=(아이템의 전체 넓이 - 마지막 아이템 넓이) 로 계산됩니다
