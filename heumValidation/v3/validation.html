heumValidation

플러그인을 통하여 간편하게 javascript 검사를 할 수 있습니다

파일다운로드

Validation

heumValidation은 submit하기전 폼들을 검사한다. 만약 validation이 하나라도 false일 경우 전송이 안됩니다




Option checkbox나 radio는 같은 name의 첫번째 요소의 옵션만 인식합니다

Data Type Default Description
data-min-length Integer 0 최소로 입력해야할 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-max-length Integer 0 최대 입력할 수 있는 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-null boolean true 빈칸 허용 할지의 유무
data-number boolean false 숫자만 사용할지의 유무
data-email boolean false 이메일 유무
data-phone boolean false 연락처 유무
data-url boolean false 인터넷 주소 유무
data-pattern String undefined 정규표현식
data-message String undefined 출력 오류 문구
