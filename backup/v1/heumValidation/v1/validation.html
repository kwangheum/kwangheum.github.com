heumValidation

플러그인을 통하여 간편하게 javascript 검사를 할 수 있습니다

파일다운로드

Validation

heumValidation은 submit하기전 폼들을 검사한다. 만약 validation이 하나라도 false일 경우 전송이 안됩니다

<form style="margin-bottom: 50px;">
        <input type="email" id="email" class="form-control" data-heum-min="2" data-heum-max="20" data-heum-null="false" data-heum-label="이메일"/>
        <input type="text" id="text" class="form-control" data-heum-min="2" data-heum-max="12" data-heum-null="false" data-heum-number="true" data-heum-label="숫자"/>
        <input type="tel" class="form-control" data-heum-null="false" data-heum-label="연락처"/>
        <input type="text" class="form-control" data-heum-validation="false" data-heum-label="검사안함"/>
        <input type="checkbox" name="checkbox" data-heum-min="1" data-heum-null="false" data-heum-label="체크1" data-heum-group-label="체크박스"/>
        <input type="checkbox" name="checkbox" data-heum-label="체크2"/>
        <input type="radio" name="radio" data-heum-label="라디오1" data-heum-group-label="라디오박스"/>
        <input type="radio" name="radio" data-heum-label="라디오2"/>
        <div class="form-group pull-right" style="padding-right: 20px;">
                <div class="btn-group">
                        <button type="submit" class="btn btn-primary">전송</button>
                        <button type="reset" class="btn btn-warning">초기화</button>
                </div>
        </div>
</form>

Option checkbox나 radio는 같은 name의 첫번째 요소의 옵션만 인식합니다

Data Type Default Description
data-heum-min Integer 0 최소로 입력해야할 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-heum-max Integer 0 최대 입력할 수 있는 글자 수 입니다, checkbox는 최소 몇개 선택
0이면 체크 하지 않습니다
data-heum-label String null 해당 요소의 라벨 이름
data-heum-group-label String null radio 또는 checkbox일때 묶일 그룹의 라벨 이름
data-heum-null boolean true 빈칸 허용 할지의 유무
data-heum-number boolean false 숫자만 사용할지의 유무
data-heum-validation boolean true 해당 요소를 검사할지 말지의 유무

Events

$("form").heumValidation().reset()
Event return Description
reset() undefined 선택된 validation 안의 내용을 초기화 시킵니다
parseBoolean(child element) boolean parseBoolean
emailValidation(child element) boolean 해당 요소가 이메일이 유효한지 검사합니다
phoneValidation(child element) boolean 해당 요소의 연락처가 유효한지 검사합니다
isEmpty(child element) boolean 해당 요소가 빈값인지 체크합니다
validations(child element) undefined 해당 요소를 검사합니다
changeView(element,division,text) undefined division에 따라 요소 색이 변경됩니다 division 종류는 error,warning,success가있습니다
text는 원하는 validation 글자를 표시합니다
(해당 기능은 jquery.bootstrap.heum.validation.js에만 해당됩니다)
